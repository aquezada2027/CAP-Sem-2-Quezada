Growth

Tangled Character Quiz!

This project represents an area of growth for me because it shows that I was able to apply the coding skills learned in class to create an interactive quiz that catered to the user’s input. The character a user receives is based on the options they choose throughout the quiz, which required me to use logic and organization in my code. The hardest part of creating the project was evenly distributing the “points” for each character, since I struggled to make the results balanced and eventually had to come up with random point values. However, the lessons from class directly influenced my success because whenever I encountered an error or bug, I usually knew how to fix it since we had discussed similar problems during class activities and assignments.
