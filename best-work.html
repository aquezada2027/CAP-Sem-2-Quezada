Best Work

This project is an example of my best work because it demonstrates how I was able to apply the coding skills and concepts learned in class to create an interactive quiz that responds to the user’s choices. I am especially proud that the quiz can generate different character results depending on the answers selected, because it shows creativity and problem-solving within my code. What makes this work stand out is the amount of logic and organization required to make the quiz function correctly. Although the project was difficult at times, especially when trying to evenly distribute the “points” for each character result, I was able to overcome those challenges by experimenting with different values and debugging my code. In order to create this project, I used skills such as coding logic, troubleshooting, problem-solving, and the knowledge I gained from class lessons, which helped me fix errors and bugs whenever they appeared.
